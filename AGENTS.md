# Matthewopoly Repo Instructions

- After every commit in this repository, push the commit to `origin/main` so the GitHub Pages site updates from the latest source.
- Before committing, check `git status --short --untracked-files=all` and avoid staging ignored QA artifacts such as `.playwright-mcp/`, `.DS_Store`, and `matthewopoly-*.png`.
- The published site is served from the `main` branch root at `https://misc111.github.io/Matthewopoly/`.
